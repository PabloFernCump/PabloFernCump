# 🚀 Proyectos destacados

## 🎾 Plataforma de reservas deportivas (Padel & Tenis)
Aplicación web full stack para la gestión de reservas en clubes deportivos.

⚛️ Frontend desarrollado con React para una experiencia dinámica e interactiva.

### 🔧 Tecnologías:
- Node.js, TypeScript, React  
- MySQL  
- Docker (docker-compose)  
- CSS

### ⚙️ Funcionalidades:
- Sistema de reservas de pistas  
- Gestión de usuarios, reservas y pistas 
- Envío automático de emails transaccionales  
- Integración de pagos online con Stripe  
- Generación de eventos añadibles a calendario (iOS / Google Calendar)  
- Dashboard con métricas y analítica del club

### 🐳 DevOps:
- Entorno dockerizado (Node + base de datos)
- Gestión de servicios con docker-compose

- 📩 Notificaciones:
  - Confirmaciones por email  
  - Integración con calendario mediante eventos descargables  

- 💳 Pagos:
  - Integración completa con Stripe para gestión de reservas  

- 📊 Analítica:
  - Visualización de datos clave del negocio mediante dashboard
 
📌 Proyecto desarrollado simulando un entorno real de negocio, enfocado en la experiencia de usuario y la gestión operativa de clubes deportivos.
